Расширение "Мониторинг неполадок"

Данное расширение анализирует датчики, имена которых начинаются с EXT, и отображает активные неполадки.

Функциональность

  • Периодически (раз в 30 сек) запрашивает список ТС через API /api_v2.php?action=get_units.
  • Отслеживает два типа неполадок: "ТС в движении" (датчик "EXT ТС в движении") и "Низкий уровень топлива" (датчик "EXT Низкий уровень топлива").
  • Неполадка считается активной, если значение датчика (hum_value) равно "вкл.".
  • В левой панели навигации отображается список типов неполадок с количеством ТС, у которых данная неполадка активна.
  • При клике на тип неполадки справа показывается подробный список ТС с указанием значения датчика и времени последнего изменения.

Используемые API

/api_v2.php?action=get_units — получение списка ТС и статусов датчиков.

Примечание

Логика определения активной неполадки по значению "вкл." может быть изменена под конкретные настройки датчиков. В коде это место отмечено комментарием.
