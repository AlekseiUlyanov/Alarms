Расширение "Alarms" (Мониторинг неполадок)

Данное расширение анализирует датчики, имена которых начинаются с определенного префикса, и отображает активные неполадки.

Функциональность

  • Периодически (раз в 30 сек) запрашивает список ТС через API
  • В левой панели навигации отображается список типов неполадок с количеством ТС, у которых данная неполадка активна.
  • При клике на тип неполадки справа показывается подробный список ТС с указанием значения датчика и времени последнего изменения.
